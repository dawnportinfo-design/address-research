# Address Infrastructure Economics

Address infrastructure has public-good characteristics, network effects, governance risk, and standards competition.

## Research Tasks

1. Model incentives for governments, carriers, EC platforms, researchers, and open-source maintainers.
2. Compare public dataset, protocol, API, and commercial validation business models.
3. Identify where fees help sustainability and where they harm adoption.
4. Define grant-ready public benefit claims.

## Safety Boundary

Economic claims should be marked as hypotheses until measured.
