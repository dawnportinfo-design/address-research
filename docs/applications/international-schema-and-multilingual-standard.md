# International Schema and Multilingual Standard

Address translation is not ordinary sentence translation. It is structure-preserving conversion across language, script, jurisdiction, and use case.

## Research Tasks

1. Define canonical field mapping by country model.
2. Distinguish translation, transliteration, normalization, and formatting.
3. Add confidence, warnings, unverified fields, postal-code match, and delivery risk.
4. Build multilingual test cases without exposing private addresses.

## Safety Boundary

Use public administrative names, synthetic examples, and redacted components.
