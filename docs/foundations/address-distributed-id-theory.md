# Address Distributed ID Theory

Address identifiers need lifecycle, uniqueness, revocation, delegation, and governance. DID methods may help, but address data must not be placed on public ledgers as raw address material.

## Core Question

How should address identifiers remain unique, portable, revocable, and governance-aware?

## Research Tasks

1. Distinguish AGID, PID, AOID, alias, commitment, and receipt.
2. Define public, private, local, and application-specific identifier scopes.
3. Define revocation and successor relationships.
4. Evaluate DID and verifiable credential compatibility without exposing address text.

## Safety Boundary

Public identifiers must not be reversible into private location data.
