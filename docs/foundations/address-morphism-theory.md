# Address Morphism Theory

Address Morphism Theory studies how surface address expressions, jurisdictional models, evidence sets, and stable identifiers map to address referents.

## Core Question

How can address forms map to stable referents without pretending every input is fully resolvable?

## Key Objects

- Surface expression: what a person, form, label, or system writes.
- Candidate set: possible referents consistent with the expression and evidence.
- Referent: the target place, entity, or delivery capability being identified.
- PID: a protocol identifier issued only when resolution is safe enough.
- History graph: the record of naming, boundary, and evidence changes.

## Research Tasks

1. State strict axioms for morphisms, candidate sets, and PID issuance.
2. Define unresolved, ambiguous, deprecated, and disputed states.
3. Prove preservation properties for safe transformation.
4. Maintain compatibility with privacy proof layers without claiming ZK can fix bad resolution.

## Safety Boundary

AMT examples should use abstract nodes and commitments. Privacy-sensitive examples belong outside public fixtures.
