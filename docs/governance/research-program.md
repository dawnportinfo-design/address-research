# Research Program

This repository should mature into a public research program with clear layers, testable claims, and safe publication rules.

## Near-Term Milestones

1. Build a complete literature and systems survey.
2. Define shared terminology across AGID repositories.
3. Add compatibility matrices for AMT, ZK predicates, country packs, and SDK contracts.
4. Publish raw-address-free benchmark and fixture policies.
5. Create grant-facing summaries for public-interest infrastructure.

## Publication Rules

- Separate verified claims from hypotheses.
- Keep political claims out of technical region identifiers.
- Prefer synthetic, aggregate, public-domain, or metadata-only examples.
- Document unsupported cases instead of hiding them.
