# Universal Address ID Namespace Design

Address identifiers need clear scope boundaries. Not every identifier should be global, permanent, public, or reversible.

## Research Tasks

1. Define AGID, PID, AOID, alias, receipt, commitment, and application IDs.
2. Separate country, region, ocean, natural geography, machine handoff, and private scopes.
3. Define collision, deprecation, successor, and revocation policies.
4. Evaluate variable-length codes and error-resistant encodings.

## Safety Boundary

Do not publish identifiers that can be reversed into private locations.
